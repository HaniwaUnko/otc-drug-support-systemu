# 🩺 OTC医薬品選択支援システム

薬学生による、OTC医薬品（一般用医薬品）を安全に提供するための選択支援システムです。  
利用者の問診情報（年齢・性別・症状・服用中の薬やサプリ・アレルギー・生活習慣など）をもとに、  
禁忌の組み合わせを自動でチェックし、適切な医薬品を提案します。

---

## 🔧 機能一覧

- ✅ 問診票フォームによるリスク情報の入力
- ✅ 禁忌情報（妊娠・授乳・肝疾患・アレルギー）に基づいた医薬品フィルタリング
- ✅ 服用中の薬やサプリメントとの相互作用の除外処理
- ✅ PMDAが公開するXMLデータのスクレイピング＆自動変換（Python）
- ✅ JSON形式の簡易データベース（db.json）による検索結果提供
- ✅ Reactによるユーザーインターフェース
- ✅（将来的に）薬剤師確認・店舗在庫連携・ネット注文の実装も想定
